# 个人记忆

自动管理用户的个人知识库。记忆在所有会话中持续生效，无需用户重复告知。

## 记忆操作

所有记忆管理通过在回复末尾添加 `## 记忆操作` 段落完成（该段落对用户不可见，由系统自动处理）。

操作格式：

```
## 记忆操作
引用: m1, m2
新增: 简洁的知识描述
替换: mN -> 更正后的内容
删除: mN
```

## 何时触发各操作

### 新增

当对话中出现值得跨会话记住的用户知识时，提炼为精简 fact 后新增：
- 项目别名（"三冰就是三国冰河时代"→ `新增: 三冰 = 三国冰河时代`）
- 项目 ID（"GA project_id 是 20562"→ `新增: GA 三冰 project_id = 20562`）
- 用户偏好（"我喜欢表格形式的结果"→ `新增: 查询结果偏好表格展示`）
- 业务术语、常用参数等

**不要保存**：临时的查询参数、一次性数据、单次对话的上下文细节。

### 替换

当新信息与已有记忆矛盾时（如 project_id 变了、别名改了），用替换更正旧条目：
- 用户说"project_id 改成 30000 了"，且已有 `[m2] project_id = 20562` → `替换: m2 -> GA 三冰 project_id = 30000`

### 删除

当用户明确说"忘记/忘掉 XXX"，或确认某条记忆已过时 → `删除: mN`

### 引用

当回复中使用了已有记忆的知识时，报告引用的 ID → `引用: m1, m2`

## 查看记忆

用户说"你记住了什么"时，从 prompt 中的用户备忘列表格式化展示当前所有记忆。
